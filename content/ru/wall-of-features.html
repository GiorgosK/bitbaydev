Стена Функций

Ниже можно ознакомиться с тем, что мы с любовью называем "Стена Функций" BitBay.
Вы всегда сможете найти ответы на любые дополнительные вопросы на нашем форуме сообщества.

 

  • Нерушимые контракты: Пиринговая Сеть, без разрыва блокчейна
  • BitBay использует знаменитый двухсторонний эскроу Halo двойного депозита
  • Децентрализованные Анонимные Рынки: Пиринговая сеть; основанная на Бит-мессенджере; потенциально более сильная, чем Tor; бессерверная
  • Доказательство Доли Владения 3.0 (PoS 3.0): обеспечивает быстрые транзакции, низкую комиссию и безопасность при этом сохраненяя сеть
  • децентрализованной и устраняя слабости Proof of Work Биткоина.
  • Кошельки с мультиподписью и с двойными паролями, скрытые внутри изображений PNG / JPG с использованием стеганографии
  • Удобные для Пользователя Шаблоны для контрактов
  • Нотариус: Отправляйте сообщение другим пользователям с нотариально заверенным текстом и хэшем документов
  • Экзотические Траты: Осуществляйте замороженные временем платежи за заморозки, коммутаторы и атомные сделки
  • Шаблон Пользовательских Контрактов: Осуществляйте микроторговлю, как настоящая децентрализованная биржа, используя BitBay в качестве поручителя
  • Контракт кранов в сфере обучения руководимая сообществом
  • Нет комиссий. Здесь все было разработано с учетом Вашей свободы мысли
  • Полная Система Переговоров по предложению / встречному предложению
  • Панель Эскроу с множеством опций, анонимный чат
  • Системы Обратной связи и репутации, проверка успешных контрактов, пост комментариев
  • Децентрализованные ключи модерации, доступные доверенным пользователям
  • Запустите свой собственный рынок, полностью анонимный
  • Холодный Стекинг с использованием двух компьютеров, невозможно взломать совместную стекинг учетную запись
  • Децентрализованное Голосование: Полностью доверительное голосование с пользовательским стекингом
  • Автоматизированные Cовместные учетные записи и возможность подписывать транзакции с помощью двух компьютеров (сильная защита).
  • Поделитесь учетной записью с партнером"
  • Шифрование / Расшифрование сообщений с использованием Ваших личных и публичных ключей
  • Отслеживание цен, чтобы заказы всегда поддерживали цену
  • Автоматически конвертируется в доллары США, следовательно Вам не нужен калькулятор
  • Возможность скрыть любой файл в изображении
  • Полная интеграция с Вашей электронной почтой, IMAP, полностью автоматизированное согласование контрактов, полностью зашифрован
  • Бит-Мессенджер, полностью интегрированная, бесплатная, децентрализованная электронная почта с каждой учетной записью
  • Оплата с помощью системы опроса в электронной почте: скрыть платежи в изображениях прямо по электронной почте
  • Anti-Keylogger: специальная клавиатура на экране для окончательного сокрушения кейлоггеров. Частные ключи перемешаны случайным образом с помощью движений мыши
  • Параметры Tor и прокси для интернет-запросов
  • Расширенная отправка с несколькими вариантами оплаты
  • Простая в использовании панель учета; взаимодействует со всеми Вашими входами
  • Параметры подписи / верификации документа и текста, используя открытые ключи, полезные для нотариуса
  • Переводит на 90 языков
  • Обучающие Видео
  • Live Python Debugger (отладчик): Запустите интерпретатор в программном обеспечении
  • IRC-чат, встроенный в программное обеспечение
  • Уникальная система управления изменениями, которая ломает все изменения в деноминации. Идеально подходит для управления множеством контрактов
